REQUIREMENTS:-
  - A C++ compiler that supports C++14 or later (g++ recommended)
  - A terminal (Linux / macOS / Windows with MinGW or WSL)



STEP 1 — FOLDER STRUCTURE:
Make sure your files are arranged like this:

  project/
  ├── main.cpp
  └── headers/
      ├── lexer.hpp
      ├── parser.hpp
      ├── ast.hpp
      ├── chunk.hpp
      └── vm.hpp




STEP 2 — COMPILE:
Open a terminal, navigate to the project folder, and run:

  g++ -std=c++17 main.cpp -o cvm

This creates an executable called "cvm" (or "cvm.exe" on Windows).



STEP 3 — RUN MODES:
1. INTERACTIVE REPL (type and run code line by line)

   Command:
     ./cvm

   Example session:
     cvm> let x = 10;
     cvm> print x;
     10
     cvm> exit


2. RUN A SCRIPT FILE
   
   Write your code in a file with the .cvm extension, then:

     ./cvm myscript.cvm

   Example script (myscript.cvm):
     let x = 5;
     let y = 3;
     print x + y;

3. DEBUG FLAGS (optional — shows internals)

   Show the token list:
     ./cvm myscript.cvm --tokens

   Show the Abstract Syntax Tree (AST):
     ./cvm myscript.cvm --ast

   Show the compiled bytecode:
     ./cvm myscript.cvm --bytecode

   Show everything at once:
     ./cvm myscript.cvm --all



STEP 4 — CVM++ LANGUAGE CHEAT SHEET:
  Declare a variable:     let x = 10;
  Reassign a variable:    x = x + 1;
  Print output:           print x;
  Take user input:        let x = input;
  If / else:              if (x > 5) { print x; } else { print 0; }
  While loop:             while (x < 10) { print x; x = x + 1; }
  Operators:              + - * /  == > < >= <=
  Comments:               // this is a comment





EXAMPLE PROGRAM — countdown.cvm

  let x = 5;
  while (x > 0) {
      print x;
      x = x - 1;
  }
  print 0;

Run it with:
  ./cvm countdown.cvm
  
NOTE:
On Windows with MinGW, replace "./cvm" with "cvm.exe":
  g++ -std=c++17 main.cpp -o cvm.exe
  cvm.exe myscript.cvm
